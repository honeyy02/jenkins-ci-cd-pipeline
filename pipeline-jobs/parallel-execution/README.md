# Parallel Job Execution in Jenkins

## 📌 Overview
This pipeline demonstrates how to execute multiple Jenkins jobs in parallel and then trigger dependent jobs sequentially.

---

## 🚀 Workflow
1. Job A and Job B run in parallel
2. After completion, Job C is triggered

---

## 🔧 Jobs Used
- Python_Job
- Simple_Maven_Project_Pipeline
- Ansible_Job

---

## 📌 Key Concept
- Parallel execution reduces build time
- Useful for microservices or independent tasks
