# Parameterized Jenkins Pipeline (Java Build)

## 📌 Overview
This pipeline demonstrates how to use parameters in Jenkins to dynamically control the build process.

---

## 🚀 Parameters
- BUILD_TRIGGER → Name of the person triggering the build
- BRANCH_NAME → Git branch to build
- SOURCE_FILE → Java file to compile and run

---

## 🔄 Workflow
1. Accept user input parameters
2. Compile Java source file
3. Execute compiled class

---

## 📌 Key Concept
- Dynamic pipelines using parameters
- Flexible build configuration
